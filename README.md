# Vocabulary Builder App

A simple and clean personal web application built with **React** to help me practice and memorize English vocabulary effectively.

## Features

* **Add New Words:** Custom form to register English words/phrases and their Spanish translations.
* **Quiz System:** A clean, distraction-free interface to practice your saved word list.
* **Score Tracker:** Real-time tracking of your correct and incorrect answers.
* **Modern Design:** Responsive layout with a focus on typography (**Lexend**) and smooth UI.

---

## 🛠️ Quick Start

Follow these steps to run the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/marialgoliva/App-Vocabulario---React.
   ```

2. **Install dependencies:**

    ```bash
    npm install
    ```
3. **Run the development server:**

    ```bash
    npm run dev
    ```
